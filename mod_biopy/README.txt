mod_biopy is installed with the following
conda create -n mod_biopy
conda activate mod_biopy
pip install git+https://gitlab.ebi.ac.uk/riziotis/biopython.git
conda install -c conda-forge pandas
conda install -c conda-forge scipy
conda conda install -c conda-forge requests

This contains the PDB folder from Biopython. it includes the version of the DSSP.py script which I modified
